# Development and Contribution

# Install Dependencies

`$` `npm install`

## Build and Serve Locally

`$` `ng serve --open`

## Lint the Code

`$` `ng lint`

## Run Unit Tests

`$` `ng test --watch=false --browsers=ChromeHeadless`
